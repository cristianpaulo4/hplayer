## ✨ Características Principais

O **HPlayer** não é apenas mais um player. Ele foi desenhado para oferecer uma experiência visual premium aliada a um processamento de mídia robusto:

- 💎 **Interface Glassmorphism**: Design moderno com efeitos de transparência (Blur) e controles flutuantes.
- 🚀 **Hardware Acceleration**: Suporte nativo para codecs pesados como **HEVC/H.265** via `media_kit`.
- 🔄 **Conversor de Formatos**: Sistema integrado de conversão de vídeo utilizando **FFmpeg**.
- ⌨️ **Controle Total**: Atalhos de teclado (Espaço para Play/Pause, Setas para Seek, F para Tela Cheia).
- 🖱️ **Interface Inteligente**: Ocultação automática de controles e cursor após inatividade do mouse.
- 🪟 **Integração Windows**: Gerenciamento de janelas nativas, incluindo modo tela cheia real e bordas customizadas.

---

## 🛠️ Stack Tecnológica

- **Framework**: [Flutter](https://flutter.dev) (Estável)
- **Motor de Vídeo**: [Media Kit](https://media-kit.dev) (Performance Nativa)
- **Processamento de Mídia**: [FFmpeg Kit](https://github.com/tanersener/ffmpeg-kit)
- **Gerenciamento de Janela**: `window_manager`
- **Estilização**: Vanilla Flutter com componentes de design moderno (Glassmorphism).

## 🚀 Como Executar

Para rodar o projeto localmente em sua máquina Windows:

1. **Pré-requisitos**:
   - Flutter SDK instalado.
   - Suporte para Desktop habilitado (`flutter config --enable-windows-desktop`).
   - Visual Studio 2022 com carga de trabalho "Desenvolvimento para Desktop com C++".

2. **Instalação**:
   ```powershell
   git clone <url-do-repositorio>
   cd hplayer
   flutter pub get
   ```

3. **Execução**:
   ```powershell
   flutter run -d windows
   ```
